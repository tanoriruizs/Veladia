# Ficha para la Chrome Web Store

## Nombre
Veladia — Detector de phishing local

## Descripción breve (132 caracteres máx.)
Detecta phishing y sitios sospechosos en tiempo real. 100% local: tu navegación nunca sale de tu equipo.

## Descripción completa
Veladia analiza cada sitio que visitas y te avisa si muestra señales de phishing, todo **sin enviar tus datos a ningún servidor**.

Cómo te protege:
• Puntuación de riesgo de 0 a 100 con un veredicto claro: seguro, sospechoso o peligroso.
• El icono cambia de color según el riesgo de la página actual.
• Banner de advertencia en sitios peligrosos para que no introduzcas tus contraseñas.
• Detecta dominios falsos (typosquatting), formularios que roban credenciales, ausencia de HTTPS, suplantación de marca y más.

Privacidad primero:
• Análisis 100% local en tu navegador.
• Cero telemetría, cero rastreadores, cero peticiones externas.
• Tú controlas la sensibilidad y tus propias listas de sitios de confianza o bloqueados.

Veladia es una capa de apoyo: no sustituye las buenas prácticas de seguridad, pero te da una señal clara antes de que sea tarde.

## Categoría
Productividad / Herramientas

## Idioma
Español

## Justificación de permisos (para la revisión)
- storage: guardar ajustes y resultados temporales localmente.
- tabs: obtener la URL de la pestaña activa para analizarla.
- scripting + host (http/https): leer el contenido de la página para detectar señales de phishing. No se transmite ningún dato.

## Práctica de privacidad declarada
No se recopilan datos de usuario. Todo el procesamiento es local.
